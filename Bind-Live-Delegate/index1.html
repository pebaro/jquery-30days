Bind, Live, Delegate

These are all deprecated and now replaced with the .on method

more information on this can be found at: jQuery uncompressed framework reference.
by running a search on this page using Ctrl + F and then entering the following in the search box: return arguments.lengthyou will be able to find which methods are attached to the .on method and also which are also attached to the .off method

Second Example Using Above Search:

jQuery.each( ( "blur focus focusin focusout load resize scroll unload click dblclick " +
	"mousedown mouseup mousemove mouseover mouseout mouseenter mouseleave " +
	"change select submit keydown keypress keyup error contextmenu" ).split( " " ),
	function( i, name ) {

	// Handle event binding
	jQuery.fn[ name ] = function( data, fn ) {
		return arguments.length > 0 ?
			this.on( name, null, data, fn ) :
			this.trigger( name );
	};
} );
